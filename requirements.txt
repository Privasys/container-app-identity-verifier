# Document OCR (GPG45 box 3 + the pre-NFC /read-mrz chip-key read): the data-page
# image is OCR'd IN THE ENCLAVE (the on-device camera OCR is unreliable on OCR-B),
# then cross-referenced against the chip. OmniMRZ wraps PaddleOCR for robust MRZ
# reading + ICAO-9303 validation. PaddlePaddle (CPU) is installed from its own
# index in the Dockerfile; the OCR models are baked in at build time so the
# enclave needs no network at runtime.
#
# NOTE: OmniMRZ is VENDORED at ./omnimrz (Apache-2.0). Its published PyPI wheel
# 0.2.1 — and a git/source install — ship metadata only, no code (the upstream
# pyproject `packages.find include` filter discards the package), so `pip install
# omnimrz` resolves but `import omnimrz` fails. We copy the source instead.
